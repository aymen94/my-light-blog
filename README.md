# my-light-blog
easy blog with nodejs, expressjs, mysql, materializecss

# all the code is commented

## Technologies and framework used
- expressjs
- mysql
- materializecss
- helmet
- ejs

## Features

- Create/Edit/Delete Post/Category/User
- express-session Authentication
- crypto-js Password Hash
- helmet security 
- materializecss ui framework 
- view engine ejs

## Server Side Dependencies

````
    "body-parser": "^1.18.3",
    "cookie-parser": "^1.4.3",
    "crypto-js": "^3.1.9-1",
    "ejs": "^2.6.1",
    "express": "^4.16.3",
    "express-session": "^1.15.6",
    "helmet": "^3.13.0",
    "mysql": "^2.16.0"
````

## Installation

### 1. Clone the repo

    git clone https://github.com/aymen94/my-light-blog.git
    
    or
    
    fork

### 2. Install Dependencies

    npm install
